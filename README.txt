Лабораторная работа №1 (Go)

Задания
1. Программа, которая выводит текущее время и дату.
2. Создание переменных различных типов (int, float64, string, bool).
3. Использование краткой формы объявления переменных.
4. Арифметические операции с двумя целыми числами.
5. Функция для вычисления суммы и разности двух чисел с плавающей точкой.
6. Программа для вычисления среднего значения трёх чисел.

Инструкции по запуску
1. Установите [Go](https://go.dev/dl/).
2. Склонируйте или скачайте проект в папку.
3. Для запуска программы используйте:
   
   go run lab1.1go
   go run lab1.2.go
   go run lab1.3.go
   go run lab1.4.go
   go run lab1.5.go
   go run lab1.6.go

go build lab1.1go
